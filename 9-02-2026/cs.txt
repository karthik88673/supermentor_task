Explanation – Client Server Architecture

Client-server architecture is a network model where the client requests data and the server provides data.

Client → User computer / mobile / browser
Server → Computer that stores website files
Internet → Network connecting client and server

Example:
When we open Google, our browser sends request to Google server, and server sends webpage.

What happens when you open a website
User enters website URL in browser
Browser sends request to server using HTTP / HTTPS
DNS finds server IP address
Request goes through internet to server
Server processes request
Server sends response (HTML, CSS, JS, images)
Browser receives response
Browser displays website on screen
